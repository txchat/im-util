## protocol wallet gen

生成钱包

### Synopsis

通过指定用户数量，能够生成相应数量的用户wallet。
生成的wallet包含助记词、公私钥、地址信息，这些信息可以通过指定写入文件和分隔符保存在文件中。
同时可以通过读取文件并指定分隔符来将用户wallet从文件中读出。
注意：当指定的数量n大于读取的用户数量时，会自动生成新的用户（填充满不足的用户）；而指定的数量n小于读取的用户数量时则以读取的为准。
每次写入时都是覆盖写入。

```
protocol wallet gen [flags]
```

### Examples

```
gen -n 1 -d
```
